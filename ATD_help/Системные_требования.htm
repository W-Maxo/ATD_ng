Системные требования

  • Процессор
     Для работы требуется компьютер с процессором Pentium с тактовой частотой не ниже 233 МГц.
  • Операционная система
     Работа возможна на следующих операционных системах:
    • Microsoft Windows Seven
    • Microsoft Windows Vista
    • Microsoft Windows Server 2008
    • Microsoft Windows Server 2003
    • Microsoft Windows XP
    • Microsoft Windows 2000 с пакетом обновления 3 (SP3) или более поздним

Поддерживаются как 32 - битные, так и 64 - битные ОС.

Microsoft Windows 98 и более ранние версии Microsoft Windows НЕ ПОДДЕРЖИВАЮТСЯ .

  • Оперативная память
     Рекомендует использовать компьютер, имеющий не менее 128 МБ оперативной памяти.
  • Свободное место на жестком диске
    Требования к наличию свободного места на жестком диске зависят от используемой конфигурации. На 1 час работы требуется около 1,1 мега байта.
  • Дисковые накопители
    Для установки требуется устройство для чтения компакт-дисков (или совместимое с ним устройство для чтения DVD-дисков).
  • Монитор
     Необходим монитор Super VGA с разрешением 800x600 или более высоким, отображающий 256 и более цветов.
  • Видео адаптер
    Для увеличения скорости выполнения операций с графикой необходимо установить видеоадаптер с аппаратным видеоускорителем.
  • Указывающие устройства
     Необходима мышь или совместимое с ними указывающее устройство.     
  • Устройства ввода
    Клавиатура.
  • Прочие требования
    • USB - порт для подключения устройства (USB 1.1, 2.0).
    • Для работы некоторых компонентов необходимы дополнительные условия. Мультимедиа:
       Для воспроизведения звуковых и других мультимедийных эффектов требуется компьютер, оснащенный средствами мультимедиа. Устройство вывода звука.
